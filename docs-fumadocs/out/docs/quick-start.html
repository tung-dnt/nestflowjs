Introduction

NestflowJS is a decorator-based state machine library for NestJS. Define workflows, handle events, and let the orchestrator drive state transitions automatically.

Workflow Definition

The @Workflow decorator is the single source of truth for your state machine - what states exist, how they connect, and what entity service handles persistence.
